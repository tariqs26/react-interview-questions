## Available Commands

| Command           | Description                                                           |
| ----------------- | --------------------------------------------------------------------- |
| `npm i`           | Install dependencies                                                  |
| `npm run dev`     | Run development server [http://localhost:3000](http://localhost:3000) |
| `npm run build`   | Build for production                                                  |
| `npm run lint`    | Run eslint                                                            |
| `npm run preview` | Run production server                                                 |
